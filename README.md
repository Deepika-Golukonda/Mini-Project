# Mini-Project
Student record system
This is a beginner-friendly Python console application that manages student information using basic data structures.
The project demonstrates fundamental backend logic such as data storage, search operations, loops, and function-based program structure.
It is suitable for learners preparing for open-source contributions or internships (e.g., GSoC, Oppia backend tasks)
Features

Add Student: Stores student details (name, age, course) in an in-memory data structure.

View All Students: Displays all stored records.

Search Student: Allows searching by student name.

Menu-Driven Interface: Runs continuously until the user chooses to exit.

Technical Concepts Practiced

Python functions
